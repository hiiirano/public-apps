# 🧺 pantry-tracker — 家庭の在庫管理

家にある食材・日用品を、**消費期限**と**残量**で管理する静的Webアプリ。

- **サーバー/ログイン/決済 不要** — データはこの端末のブラウザ（localStorage）にのみ保存
- **消費期限の近い順**に並び、期限切れ・3日以内が色で分かる
- **最低在庫**を割った物が「残量わずか」として買い物の合図になる
- カテゴリ／保管場所／要対応のみ／品名で絞り込み
- 任意で **LLM拡張**：レシートやメモを貼り付けて在庫を自動取り込み（APIキーは端末内のみ・未設定でも全機能利用可）

`meal-planner` の姉妹アプリ。設計思想（決定的コアはLLM無しでフル動作・LLMは任意の上乗せ）を踏襲。

## 開発

```bash
npm install
npm run dev      # 開発サーバ
npm test         # コアロジックのユニットテスト
npm run build    # 型チェック＋本番ビルド（dist/）
```

## 構成

| ファイル | 役割 |
|---|---|
| `src/types.ts` | 型定義・カテゴリ/単位・しきい値 |
| `src/inventory.ts` | 決定的コア（緊急度・並べ替え・フィルタ・集計／純関数） |
| `src/storage.ts` | localStorage 永続化・入力正規化 |
| `src/llm.ts` | 任意のLLM抽出（OpenAI互換・なくても動く） |
| `src/main.ts` | 素のDOMによるUI |
| `src/inventory.test.ts` | ユニットテスト |
| `SPEC.md` | 仕様 |

ランタイム依存は0。ビルド後は静的アセットのみで、任意のサブパスに置けます（`vite.config.ts` の `base: "./"`）。
