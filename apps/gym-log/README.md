# 筋トレ記録 gym-log

自重トレとジムマシン（チョコザップ標準マシンをプリセット収録）のトレーニングを、
スマホでその場で記録する小さなアプリ。アカウント不要・無料。
データは**この端末のブラウザ（localStorage）だけ**に保存されます。

公開URL: https://hiiirano.github.io/public-apps/gym-log/

## できること

- **セット記録**: 種目ごとに重量(kg)×回数を記録。前回の記録が placeholder と「前回:」表示で見えるので、伸ばすべき数字が分かる
- **今日のおすすめ**: 履歴から上半身/下半身のローテーションを提案（2日連続なら休養日を提案）
- **プリセット種目**: 自重7種＋チョコザップ標準マシン8種。追加・編集・非表示も自由
- **今週サマリー**: 回数・セット数・総ボリューム(kg)
- **PWA**: ホーム画面に追加すればアプリとして全画面起動。オフラインでも動作
- **画面スリープ防止**: 記録中は画面が消えない（設定でON/OFF、Android Chrome対応）
- **バックアップ**: JSON エクスポート/インポート、CSV エクスポート

## 開発

```bash
npm install
npm run dev    # 開発サーバー
npm test       # ロジックのユニットテスト (tsx)
npm run build  # 型チェック + ビルド
```

設計は [SPEC.md](SPEC.md) を参照。
