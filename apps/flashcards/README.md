# 暗記カード（flashcards）

テキストを貼ると **AI が Q&A カードを自動生成**し、**SM-2 間隔反復**で復習できる暗記アプリ。
ログイン不要・インストール不要。ブラウザだけで完結し、データはこの端末の localStorage にのみ保存されます。

🔗 ライブ: https://hiiirano.github.io/public-apps/flashcards/

## できること

- **デッキ管理** — カードの束を作成・選択・削除。
- **AI カード生成（主機能）** — ノートや文章を貼り付けると、任意の LLM が一問一答に変換。
  プレビューで取捨選択してから追加できる（生成を丸呑みしない）。
- **手動追加** — 表（問い）/裏（答え）を入力して 1 枚ずつ追加・編集・削除。
- **学習セッション** — 今日の対象カード（復習＋新規）を 1 枚ずつ表示 → 答えを見る →
  **4 段階の自己評価（もう一度 / むずかしい / ふつう / かんたん）**。
- **SM-2 間隔反復** — 評価に応じて次回出題日を自動計算（LLM 不要・オフライン動作）。
- **バックアップ** — JSON でエクスポート / インポート。別端末への移行に。

LLM が未設定でも、手動追加・学習・スケジュールはすべて使えます。

## 設計

決定論的なコア（間隔反復・出題抽出・集計）と、任意の LLM 拡張を分離。

- `src/types.ts` — 型・定数・サンプルデッキ seed。
- `src/srs.ts` — **決定的コア**（純関数）。SM-2 アルゴリズム・出題キュー・集計。
- `src/storage.ts` — localStorage 永続化と正規化、エクスポート/インポート。
- `src/llm.ts` — 任意 LLM 拡張（OpenAI 互換 Chat Completions）。未設定なら呼ばれない。
- `src/main.ts` — 素 DOM の UI（full re-render 方式）。

### SM-2 の概要

4 段階評価を SM-2 の quality（もう一度=2 / むずかしい=3 / ふつう=4 / かんたん=5）に対応させ、
連続正答回数・間隔（日）・易しさ係数（EF, 初期 2.5・下限 1.3）から次回出題日を決める。
「もう一度」は当セッションの末尾に再投入して取りこぼしを防ぐ（永続スケジュールは翌日）。
詳細は [`SPEC.md`](./SPEC.md)。

## 開発

```sh
npm install
npm run dev      # 開発サーバ
npm test         # コアのユニットテスト（tsx）
npm run build    # 型チェック + 本番ビルド
```

## プライバシー

- すべてブラウザ内で動作。カードもデータもサーバには送りません。
- LLM 拡張を使う場合のみ、入力テキストと API キーが設定したエンドポイントへ送信されます。
  API キーはこの端末の localStorage にのみ保存されます。
